# Final Steps in Project- 
- GitHub Actions automatically deploy your Docker container to your EC2 server after pushing the image to Amazon Elastic Container Registry.
- GitHub → Build Image → Push to ECR → Deploy to EC2
- Step 1: Locate the same key pair you used earlier 'flaskappkeypair.pem'
  - Open locally 
```
cat flaskappkeypair.pem
```
- Step 2 — Add GitHub Secret
```
Name: EC2_SSH_KEY
Value: (paste flaskappkeypair.pem contents)
Name: EC2_HOST
Value: your EC2 public IP
Name: EC2_USER
Value: ec2-user
```
- Step 3 — Update GitHub Actions Workflow
```
- name: Deploy to EC2
  uses: appleboy/ssh-action@v1.0.3
  with:
    host: ${{ secrets.EC2_HOST }}
    username: ${{ secrets.EC2_USER }}
    key: ${{ secrets.EC2_SSH_KEY }}
    script: |
      docker stop flask-app || true
      docker rm flask-app || true
      docker pull 859525218899.dkr.ecr.us-east-1.amazonaws.com/flask-devops-app:latest
      docker run -d -p 80:5000 --name flask-app 859525218899.dkr.ecr.us-east-1.amazonaws.com/flask-devops-app:latest
```
- Commit and push to trigger pipeline
