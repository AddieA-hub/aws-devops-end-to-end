# Day 1

# Install and Update Local Tools
- Git
- Docker
- Python 3
- AWS CLI
- Terraform

- Step 1
Install Github
Create the GitHub repository (10 minutes)
Go to github.com
Click New Repository
Repository name: aws-devops-end-to-end
Set to Public
Check:
 ☑️ Add README
 ☑️ Add .gitignore → choose Python
Click Create Repository

-Step 2
Open README.md and paste this exactly:
AWS DevOps End-to-End Project (include a double hash to bold all headings as needed)

 This project demonstrates a complete DevOps workflow on AWS Free Tier.

Tech Stack 
- Application: Python (Flask)
- Containerization: Docker
- Infrastructure as Code: Terraform
- CI/CD: GitHub Actions
- Cloud Provider: AWS
- Monitoring: CloudWatch

Goal
To build, deploy, and automate a containerized application on AWS using modern DevOps practices.

-Step 3
Create a local project folder
On your laptop: Create a folder: aws-devops-end-to-end
Open a terminal inside it

- Step 4
Code to Clone your repo: 
git clone https://github.com/AddieA/aws-devops-end-to-end.git
cd into the folder:
cd aws-devops-end-to-end
You should now see README.md inside.

- STEP 5: 
Check / install required tools
Run these one by one. If something isn’t installed, install it.
Check Git: git --version
Check Python: python3 --version
Check Docker: docker --version
Check AWS CLI: aws --version
Check Terraform: terraform --version
